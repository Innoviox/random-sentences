CCGbank

Julia Hockenmaier and Mark Steedman
http://groups.inf.ed.ac.uk/ccg



CCGbank is a translation of the Penn Treebank into a corpus of Combinatory Categorial Grammar derivations. It pairs syntactic derivations with sets of word-word dependencies which approximate the underlying predicate-argument structure. The translation process and linguistic analyses are explained in the manual. Additional information is given in the readme file.
CCGbank contains 99.44% of the sentences in the Penn Treebank, for which it corrects a number of inconsistencies and errors in the original annotation.

In addition to the HTML version below, there are two machine-readable versions of the data:
  • AUTO contains the syntactic derivations.
  • PARG contains the corresponding lists of word-word dependencies.
These formats are explained in AUTO readme and PARG readme files.

In the HTML files below, the syntactic derivation of each sentence is followed by the word-word dependencies which approximate the underlying predicate-argument structure.
In all versions, the file structure corresponds exactly to that of the original Treebank.
The RAW directory contains the raw text of all sections in CCGbank (excluding those sentences in the original Treebank for which CCGbank contains no derivation). These sentences do not contain any quotation marks that appear in the original Wall Street Journal text, because these do not appear in CCGbank.

CCGbank can also be searched with Douglas Rohde's TGrep2, version 1.15 or higher, if the CCGbank TGrep2 file is used as corpus. More information can be found at the TGrep2 site as well as in the TGrep2 and CCGbank readme file.
