CCGbank


Introduction

This file contains documentation on the CCGbank, Linguistic Data Consortium (LDC) catalog number LDC2005T13 and ISBN 1-58563-340-2.

Additional documentation is available at the publication providers web site at http://groups.inf.ed.ac.uk/ccg.

CCGbank is a translation of the Penn Treebank into a corpus of Combinatory Categorial Grammar derivations. It pairs syntactic derivations with sets of word-word dependencies which approximate the underlying predicate-argument structure. The translation process and linguistic analyses are explained in the manual. Additional information is given in the README file.

This updated version 1.1 corrects a misalignment of sentences between the PARG and AUTO files, as well as a problem with some original POS tags in the AUTO files. It also contains the ccgbank.00-24.t2c file which was not contained on the previous CD.

CCGbank contains 99.44% of the sentences in the Penn Treebank, for which it corrects a number of inconsistencies
and errors in the original annotation.

The human-readable HTML version can be accessed here. There are also three machine-readable versions of the data:

  • AUTO contains the syntactic derivations.
  • PARG contains the corresponding lists of word-word dependencies.
  • TGREP2 contains a file ccgbank.00-24.t2c which can be used in conjunction with Douglas Rohde's TGrep2, version 1.15 or higher.

These formats are explained in AUTO readme and PARG readme files.

In the HTML version, the syntactic derivation of each sentence is followed by the word-word dependencies which approximate the underlying predicate-argument structure. In all versions, the file structure corresponds exactly to that of the original Treebank.
